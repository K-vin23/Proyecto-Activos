# **App Name**: Activos Gestión Pro

## Core Features:

- User Registration: Collect user data (ID number, first name, middle name, last name, second last name, city, address, department, email).
- Password Creation: Require the user to create a password of at least 8 characters, including one special character.
- Form Validation: Provide visual validation and error messages for each field during registration.
- Data Submission Handler: Provide an 'onRegister' function that receives user data in an object and sends data to your database connector.
- Asset Summary Dashboard: Display a dashboard with asset summaries and charts (total assets, assets by department, alerts, and open tasks).
- Recent Notifications: Display recent notifications related to assets and system activity.
- Asset Management Tools: Offer an advanced search function and quick access buttons to register new assets, generate reports, and request technical assistance.

## Style Guidelines:

- Primary color: Dark navy blue (#1A237E) to convey professionalism and stability, reminiscent of classic management tools.
- Background color: Light gray (#F5F5F5), almost white, creating a clean and uncluttered workspace that increases readability.
- Accent color: Deep orange (#E65100) to draw the user's eye to important interactive elements and system alerts.
- Font pairing: 'Inter' (sans-serif) for body and 'Space Grotesk' (sans-serif) for headings to maintain a contemporary but readable style.
- Code font: 'Source Code Pro' for displaying any code snippets.
- Use clean, consistent icons from a library like Material Icons to represent asset categories, actions, and status indicators.
- Implement a responsive layout using Material UI's grid system to ensure adaptability to different screen sizes and devices.
- Use subtle animations and transitions to provide feedback and improve the overall user experience. Focus on meaningful transitions that enhance usability, such as expanding details on selection.
